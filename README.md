Evaluación final módulo 4 

Objetivos de la aplicación:
Aplicación web que consume API (https://jsonplaceholder.typicode.com/users) usando XMLHttpRequest.
Funcionalidades: 
1. Listar los nombres de todos los usuarios
2. Buscar usuario por nombre y apellido -> datos básicos
3. Buscar usuario por nombre y apellido -> dirección completa
4. Buscar usuario por nombre y apellido -> datos avanzados
5. Listar compañías
6. Mostrar nombres ordenados alfabéticamente

URL del repositorio
